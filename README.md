# NextJs Stackable

## Stack

- [Next.js](https://zeit.co/blog/next)
- [React.js](https://facebook.github.io/react/)
- [Stackable.space](http://www.stackable.space)

## Setup

Run:

```
npm install
```

## Development

To start development server

```
npm run dev
```

## Deployment

```
npm run build
npm start
```
